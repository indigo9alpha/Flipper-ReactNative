Flipper(https://github.com/indigo9alpha/Flipper)のモバイルアプリ版。
React Nativeで開発しました。


## 使用技術
* ReactNative
* react-navigation
* react-native-animatable
* react-native-paper
* react-redux
* redux
* redux-thunk
* expo
* firebase
* use-count-up
